帮助 InPageEdit 完善

作为一个为编辑者造福的用户脚本，我们深知只有听取用户的意见、指导才能更加完善。

同时，作为一个正在发展、持续更新的项目，我们也深知使用过程中出现漏洞、问题是不可避的。

因此，我们采用了 Github issue 作为用户汇报意见的平台。

如何汇报？

您只需要进入 InPageEdit 的 Github repo 后，点击“Issue”，之后点击“Start a new issue”。

请务必按照模板填写好资料，之后提交即可。issue 不规范，🐟君两行泪。

在您提交之后，机智的小鱼君或其他的志愿者其实根本没有会开始研究并跟踪此 issue。尽情耐心等待结果。

或点击按钮：

直达 issue
